# TODO Laravel App

## Project Setup

> git clone <https://github.com/ZerfyT>

    Config your .env file

cp .env.example .env

> composer install

> npm install

> php artisan key:generate

> php artisan make model Todo -m

> php artisan migrate

> php artisan make:controller API/TodoController --api

    List down your Routes

> php artisan route:list

    Run the project

> php artisan serve

> npm run dev
